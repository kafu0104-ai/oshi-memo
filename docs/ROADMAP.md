# 🤍 ROADMAP

# 開発ロードマップ

推しメモは、実際の推し活で利用しながら段階的に機能を追加していきます。

機能数を増やすことではなく、
実用性を高めることを優先して開発を進めます。

# 優先順位

開発では以下の順番を重視します。

1. 実際に使えること
2. 分かりやすいこと
3. 快適に操作できること
4. 新しい機能

機能追加よりも、
既存機能の品質向上を優先します。

---

# 開発方針

推しメモは、
実際の推し活で得られた経験や課題をもとに改善を続けます。

「こんな機能が欲しかった」

という気づきを大切にしながら、
継続的にアップデートを行います。
