# 🤍 推しメモ (OSHI-MEMO)

> 推し活で「あ、忘れてた…」をなくすアプリ。

推しメモは、グッズ購入・チケット管理・立替精算・受け渡し・通知など、
推し活で発生するさまざまなタスクをイベント単位で管理できるアプリです。

単なるメモアプリではなく、

**「推し活のアシスタント」**

として、次にやるべきことを忘れないようサポートします。

---

# 👤 対象ユーザー

- 推し活をしている人
- グッズ購入数が多い人
- ライブ・イベントによく参加する人
- 立替や共同購入をする人
- チケット管理をまとめたい人

---

# ✨ 主な機能

## 🛍 グッズ管理

- 商品一覧管理
- 数量管理
- 購入者管理
- 売切れ管理
- 再入荷待ち
- 事後通販待ち
- 特典自動計算

---

## 🎫 チケット管理

- 応募管理
- 当落管理
- 支払い管理
- 発券管理
- 座席管理
- 同行者管理

---

## 💰 精算管理

- 立替管理
- 未精算管理
- 受け渡し管理
- 精算内容共有

---

## 🔔 通知

必要な人だけ通知できる設計です。

例

- 当落発表
- 支払い期限
- 発券開始
- イベント前日
- 事後通販開始
- 事後通販終了

通知タイミングも自由に設定できます。

---

## 📋 やることリスト

ホーム画面では

- 未精算
- 発券開始
- 再入荷待ち
- 事後通販待ち
- 未受け渡し

など、

**今やるべきこと**

を一覧で確認できます。

---

# 🎯 コンセプト

推し活では

- 買い忘れ
- 発券忘れ
- 支払い忘れ
- 特典の配分漏れ
- 精算漏れ

など、

小さな「忘れてた」が積み重なります。

推しメモは

**「推し活で忘れないこと」**

を第一に考えて設計しています。
